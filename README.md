shinyMILC: A Graphical User Interface to MILC
---------------------------------------------

**Not usable in it's current state.**

A very basic and experimental user interface to the MILC package by Stavroula
Chrysanthopoulou. 

Install from Github:

```
devtools::install_github("christophsax/shinyMILC")
```

and run the App:

```
library(shinyMILC)
shinyMILC()
```
